Supported Protocols

Six streaming protocols, one unified interface. Choose the right technology for your use case.

🔌

WebSocket

Full-duplex communication for real-time chat, gaming, and collaborative applications.

📡

Server-Sent Events

Unidirectional server-to-client streaming for live updates, notifications, and data feeds.

🌊

HTTP Streaming

Chunked transfer encoding for progressive data loading and streaming responses.

🔄

Long Polling

HTTP-based persistent connections for environments where WebSockets aren't available.

📺

HLS Video

HTTP Live Streaming for adaptive bitrate video with automatic quality adjustment.

🎥

WebRTC

Peer-to-peer real-time communication for video calls, file sharing, and media streaming.

Simple, Declarative API

No complex configuration or boilerplate. Just specify the protocol type and configuration object, and you're ready to stream.

The component handles connection management, error recovery, and provides reactive state updates automatically.
